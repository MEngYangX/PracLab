# 配置文件

PracLab 的配置文件全部位于服务器的 `<CS2>/csgo/cfg/PracLab/` 目录下，仓库 [`cfg/PracLab/`](../../cfg/PracLab/) 提供默认模板。修改后需重启插件或服务器生效。

## 1. `config.cfg` — 插件总开关

**路径**：`<CS2>/csgo/cfg/PracLab/config.cfg`
**格式**：每行一个 ConVar，`key value` 用空格分隔，`//` 开头为注释。

| ConVar                     | 类型     | 默认值     | 说明                                                                     |
| -------------------------- | ------ | ------- | ---------------------------------------------------------------------- |
| `praclab_enabled`          | bool   | `true`  | 插件总开关。`false` 时所有命令（含 `.prac`、`.map`、地图切换）均不响应，向发送者打印「practice 模式未启用」。 |
| `praclab_default_language` | string | `zh-CN` | 服务器默认语言代码。可选 `zh-CN` / `en`。                                           |

## 2. 录制文件目录

**路径**：`<CS2>/csgo/cfg/PracLab/recordings/`

`.record` 命令保存的 JSON 录制文件存放在此目录，文件名格式 `<Id>_<PlayerName>.json`。`.clearrecord <Id>` 与 `.clearrecordall` 会同步删除文件与内存条目。

**自动创建**：插件 `Load` 时调用 `EnsureRecordingsDir()` 确保目录存在，无需手动创建。

**地图切换清空**：`OnMapStart` 监听器会清空内存中的录制列表（`_recordings.Clear()`），但**不删除 JSON 文件**；如需保留跨地图录制，可在玩家切换地图前先用 `.currentrecord` 记录 Id。

## 3. 出生点方框地图配置

每张地图的方框 Z 偏移与序号 yaw 旋转角度硬编码在 [`Commands/SpawnMarkerCommands.cs`](../../Commands/SpawnMarkerCommands.cs) 中：

- `SpawnTextYawByMap`：每张地图每队序号文本的 yaw 旋转角度（度）
- `SpawnExtraZOffsetByMap`：每张地图的方框/文本额外 Z 抬高量

**当前配置**：

| 地图          | T yaw | CT yaw | 额外 Z 偏移 |
| ----------- | ----- | ------ | ------- |
| de\_cache   | 90    | 180    | 0       |
| de\_train   | -90   | 0      | 0       |
| de\_anubis  | 0     | 180    | 0       |
| de\_vertigo | 90    | 180    | 0       |
| de\_ancient | 0     | 180    | 16      |
| de\_nuke    | -90   | 90     | 0       |
| de\_mirage  | 90    | -90    | 0       |
| de\_inferno | -90   | 90     | 0       |

未配置的地图默认 yaw=0、Z 偏移=0。新增地图支持时需在此表中补充配置并重新编译。
