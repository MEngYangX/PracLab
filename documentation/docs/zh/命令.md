# 聊天栏命令

PracLab 通过拦截全局聊天（`say`）与队伍聊天（`say_team`）来识别玩家指令。所有命令均支持以下两种前缀形式，大小写不敏感：

| 前缀  | 示例      |
| --- | ------- |
| `.` | `.prac` |
| `!` | `!prac` |

***

## 命令总览

> **E 键传送**：执行 `.prac` 开启 practice 模式后，地图上会自动显示 CT 和 T 的竞技模式出生点方框（绿色）。玩家对准任一方框按下使用键（默认 E）即可传送到该出生点，聊天栏提示「已传送到第 N 个出生点」。可用 `.showspawns` / `.hidespawns` 手动控制显示。

### Practice 模式控制

| 命令      | 别名 | 功能                    |
| ------- | -- | --------------------- |
| `.prac` | —  | 加载练习配置，开启 practice 模式 |

### 出生点方框

| 命令            | 别名 | 功能                                            |
| --------------- | -- | --------------------------------------------- |
| `.showspawns` | —  | 重新显示双方出生点方框（prac 模式开启时自动显示）                   |
| `.hidespawns` | —  | 隐藏出生点方框                                       |

> **E 键传送**：对准方框按下使用键（默认 E）即可传送到该出生点，聊天栏提示「已传送到第 N 个出生点」。CT 和 T 方框均为绿色。prac 模式下不限制队伍，玩家可瞄准任一方框传送。方框仅显示竞技模式出生点（与 `.spawn` 系列命令一致）。

## 使用限制

- **总开关**：所有命令受 `cfg/PracLab/config.cfg` 中 `praclab_enabled` 控制（`true` / `false`）。关闭时所有命令均不响应，并向发送者打印「practice 模式未启用」。
- **Practice 模式**：除 `.prac` 外，所有命令均要求 practice 模式已激活（即已执行过 `.prac`）。未激活时向发送者打印「请先使用 .prac 开启 practice 模式」，命令不执行。
- **语言**：语言仅由 `config.cfg` 中的 `praclab_default_language` 配置项控制（默认 `zh-CN`，可选 `en`）。服务器管理员修改该值并重启插件即可切换全服默认语言。
