# PracLab

> CS2（Counter-Strike 2）练习模式插件 / A CS2 practice-mode plugin

基于 [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp) 与 [Metamod:Source](https://www.sourcemm.net/)，为竞技训练场景提供完整的道具/出生点/Bot/回放辅助命令。

Built on [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp) and [Metamod:Source](https://www.sourcemm.net/), providing a complete set of utility commands for grenade/spawn/bot/replay training scenarios.
